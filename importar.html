📂 Paso 1 — Carga tu CSV
Exporta desde tu hoja: Archivo → Descargar → CSV
📄
Arrastra tu CSV aquí
o haz clic para seleccionar
o pega el contenido directamente
🔗 Paso 2 — Mapea las columnas
Indica qué columna de tu hoja corresponde a cada campo. Si una columna no existe selecciona "— ninguna —".
👀 Paso 3 — Vista previa
Revisa que todo esté correcto antes de exportar.
📋 JSON generado
🚀 Paso 4 — Actualiza el catálogo
1
Reemplaza el archivo Copia el archivo productos.json descargado a la carpeta del proyecto (reemplaza el anterior).
2
Sube las imágenes Copia tus fotos a la carpeta imagenes/. Usa los mismos nombres que pusiste en la columna IMAGEN.
3
Sube a GitHub Abre la terminal en la carpeta del proyecto y ejecuta:
git add . && git commit -m "Actualizo productos" && git push
4
Vercel despliega automáticamente En 30 segundos tu catálogo online estará actualizado. ✅
